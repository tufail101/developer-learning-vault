# Mini Project — First Flask App

## What To Build
Build a small Flask app with a homepage and one extra page.

## What It Should Do
- start a Flask development server
- return one message at `/`
- return a different message at `/about`
- work in the browser at the local Flask address

## What You Must Use
- Flask app setup
- `@app.route()`
- `app.run(debug=True)`

## Stretch Goal
Return HTML instead of plain text for one route.
