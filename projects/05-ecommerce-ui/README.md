# Ecommerce UI

This project is a small ecommerce interface with a product grid, interactive cart sidebar, live item count, and total calculation.

## Skills It Demonstrates

- responsive product grid with `auto-fit`
- cart icon badge updates
- add to cart interactions
- remove from cart interactions
- cart total calculation
- slide-in cart sidebar and overlay toggle

## How To Open It

Open [src/index.html](/home/tufail/Dev/Study/web-dev-learning-system/projects/05-ecommerce-ui/src/index.html) in a browser.
