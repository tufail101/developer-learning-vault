# Practice — Functions

## Exercise 1
Write a function called `square_number` that takes one number and returns its square.
Call it with at least two different numbers and print the results.

## Exercise 2
Write a function called `introduce` that takes a name and city.
Make it print a sentence like `"Hi, I am Aisha from Lahore."`
Call it twice with different values.

## Exercise 3
Write a function that takes a price and a discount percentage.
Return the discounted price.
Store the returned value in a variable and print it.

## Done Checklist

- [ ] I can explain which function uses `print()` and which one uses `return`
- [ ] my discount function gives a value back instead of only printing inside the function
- [ ] I reused at least one function with different input values
