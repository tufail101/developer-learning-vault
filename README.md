# Web Dev Learning System

This repository is a mentor-style, chapter-by-chapter learning system for going from complete beginner to job-ready web developer over roughly 3 years of steady practice.

The structure is inspired by the shared HTML and JavaScript playlists, but it does not stop there. I added missing topics that matter in real work: CSS layout, responsive design, accessibility, debugging, async JavaScript, API usage, project thinking, and portfolio-building.

## Who This Is For

- Someone starting from zero
- Someone who has watched tutorials but still feels confused
- Someone who wants a roadmap instead of random videos
- Someone who wants to build projects that look like real frontend work

## What You Will Learn

1. HTML from structure to semantic markup
2. CSS from selectors to responsive layouts
3. JavaScript from variables to DOM and APIs
4. Practical UI building through guided projects
5. Backend learning with Node.js, Express, MongoDB, and MERN
6. Python basics and Python web development paths
7. Java basics and Spring Boot backend paths
8. Learning habits that help you stay consistent for years
9. Year 3 React, TypeScript, testing, and career prep

## Prerequisites

You do not need a computer science degree or past coding experience to begin this repo.

Before starting, you should have:

- a web browser
- a text editor like VS Code
- willingness to practice consistently, even when progress feels slow

Helpful but optional:

- a GitHub account
- Node.js for later JavaScript and tooling sections
- curiosity to break examples and rebuild them

## Study Philosophy

Learn like a builder, not like a collector of tutorials.

- Read a chapter
- Run the example
- Change the example
- Complete the practice work
- Finish the mini challenge
- Build the mini project
- Repeat

If you get stuck, slow down. Confusion is not failure. Confusion is usually the brain building new connections.

## Suggested Pace

- Year 1: HTML, CSS, JavaScript basics, lots of repetition
- Year 2: DOM, APIs, deeper projects, clean code habits
- Year 3: React, TypeScript, testing, portfolio polish, and interview readiness

## Year 3: Senior Track

When you finish the core frontend path, move into [senior-track/](./senior-track/).

This section is split into:

- `react/`: component architecture, state, effects, routing, and context
- `typescript/`: type safety, generics, React typing, and tooling
- `testing/`: testing strategy, Jest, DOM testing, and testable design
- `career/`: interview prep, system design basics, PR review, portfolio, and negotiation

## Output Goal

By the end of this repository, you should be able to:

- build responsive websites
- understand browser-based JavaScript
- read and modify existing code
- create portfolio-grade projects
- explain your decisions like a professional developer
