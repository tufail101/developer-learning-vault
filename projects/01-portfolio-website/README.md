# Portfolio Website

This project is a polished portfolio landing page for a frontend developer.

## Skills It Demonstrates

- semantic HTML page structure
- CSS variables for design tokens
- Flexbox navigation
- CSS Grid for skills and project cards
- responsive design with media queries
- smooth scrolling
- active navigation highlighting on scroll
- client-side contact form validation

## How To Open It

Open [src/index.html](/home/tufail/Dev/Study/web-dev-learning-system/projects/01-portfolio-website/src/index.html) in a browser.
