# Instructions — Daily Notes Saver

1. Create a file such as `daily_notes.py`.
2. Use `input()` to ask the user for one short note.
3. Open a text file like `notes.txt` in append mode.
4. Write the new note into the file with a newline at the end.
5. Open the same file in read mode.
6. Print all saved notes to the terminal.

## Suggested Build Order

1. Get the input line working first.
2. Append one note to the file second.
3. Read the file back third.
4. Add the date stretch goal last.

## What To Check Before You Stop

1. Does a new note get added without deleting old notes?
2. Does the file print cleanly after each run?
3. Did you use `with open(...)` for both reading and writing?
