# Mini Project — Interrupt And Resume Workflow

## What To Build
Practice switching away from unfinished work, handling something urgent, and then restoring your original changes.

## What It Should Do
- stash unfinished changes with a readable message
- let you switch tasks without making a fake commit
- restore the original work afterward

## What You Must Use
- `git stash push`
- `git stash list` or `git stash pop`

## Stretch Goal
Create two stashes with different messages and apply only the one you want.
