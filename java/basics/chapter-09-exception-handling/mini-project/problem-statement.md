# Mini Project — Safe Input Program

## What To Build
Build a Java program that safely handles bad number input or calculation errors.

## What It Should Do
- run risky code inside `try`
- catch at least one specific exception type
- print a friendly message when something goes wrong

## What You Must Use
- `try` and `catch`
- one specific exception type

## Stretch Goal
Add a retry flow that lets the user try again after a bad value.
