# Mini Project — First Tracked Project Folder

## What To Build
Create a tiny project folder and track its history with Git from the beginning.

## What It Should Do
- Initialize a Git repository
- Add at least two files
- Create the first commit with a clear message

## What You Must Use
- `git init`
- `git add`
- `git commit`

## Stretch Goal
Make a second commit after changing one file and compare the two commit messages.
