# Practice — Closures And Callbacks

## Exercise 1
Write a `createCounter()` function that returns an inner function and make sure repeated calls keep increasing the same value.

## Exercise 2
Create a function that accepts a callback and runs it after logging a message like `Task finished`.

## Exercise 3
Build a small example that uses both ideas together, such as a callback that logs the latest count after a button click or timeout.
