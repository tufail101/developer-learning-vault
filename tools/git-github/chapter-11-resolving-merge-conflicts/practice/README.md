# Practice — Resolving Merge Conflicts

## Exercise 1
Create two branches in a test repo, change the same line in both branches, and merge them to trigger a conflict.

## Exercise 2
Open the conflicted file, read both versions slowly, and choose one final version manually.

## Exercise 3
Run `git status` after resolving the file so you can see what Git expects next before completing the merge.
