# Mini Project — Simple Template Site

## What To Build
Build a small Flask page that renders HTML from a Jinja2 template.

## What It Should Do
- render a template file
- display data passed from the route
- loop through a short list in the template
- load in the browser as an HTML page, not plain text

## What You Must Use
- `render_template()`
- at least one template variable
- at least one Jinja loop

## Stretch Goal
Create a base template and reuse it for two pages.
