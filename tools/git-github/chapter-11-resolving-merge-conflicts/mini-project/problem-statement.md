# Mini Project — Conflict Resolution Practice Repo

## What To Build
Create a small practice repo where you trigger and resolve a merge conflict on purpose.

## What It Should Do
- create a conflict by changing the same lines on two branches
- resolve the file cleanly by hand
- finish the merge and confirm the final file still makes sense

## What You Must Use
- two branches with conflicting edits
- manual conflict resolution

## Stretch Goal
Write a short note about how you decided between the two versions instead of choosing blindly.
