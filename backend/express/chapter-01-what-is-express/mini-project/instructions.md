# Instructions — First Express Server

1. Create a new folder for the project.
2. Run `npm init -y`.
3. Run `npm install express`.
4. Create `app.js`.
5. Add the Express setup code and one `GET /` route.
6. Start the server with `node app.js`.
7. Open the home route in the browser.
8. Add the `/about` route and test it too.

## Suggested Build Order

1. install Express first
2. make `/` work second
3. add `/about` third
4. add `/contact` only after the first two routes work

## What To Check Before You Stop

1. Does the terminal show the startup message?
2. Does `/` return the home text?
3. Does `/about` return different text?
4. Can you point to the line where the server starts listening?
