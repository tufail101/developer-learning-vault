# Practice — Type Conversion And Scope

## Exercise 1
Create three variables as strings that look like numbers, then convert them with `Number()` and log the results.

## Exercise 2
Write a small function with one local variable and prove that the variable cannot be used outside the function.

## Exercise 3
Build a form-free calculator in JavaScript that starts with string values like `"5"` and `"8"`, converts them, and logs the numeric total.
