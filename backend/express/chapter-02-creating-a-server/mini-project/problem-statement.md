# Mini Project — Local Dev Server Starter

## What To Build
Build a simple Express server that is easy for you to start, test, and restart while learning.

## What It Should Do
- Run on a chosen local port
- Serve a home route response
- Print a readable startup message when the server begins listening

## What You Must Use
- `app.listen()`
- a chosen port number
- a route that proves the server works

## Stretch Goal
Move the port into a variable at the top so it is easy to change later.
