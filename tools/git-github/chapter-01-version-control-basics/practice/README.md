# Practice — Version Control Basics

## Exercise 1
Create a new practice folder, run `git init`, and confirm that Git starts tracking the folder.

## Exercise 2
Create one text file, add one line to it, and run `git status` so you can see an untracked file become visible to Git.

## Exercise 3
Stage the file with `git add`, run `git status` again, and explain the difference between untracked and staged in your own words.
