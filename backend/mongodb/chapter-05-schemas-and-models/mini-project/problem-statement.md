# Mini Project — Learning Resource Model

## What To Build
Create a Mongoose schema and model for one resource in a learning app.

## What It Should Do
- Define a schema with at least three fields
- Create a model from the schema
- Keep the field names clear and realistic

## What You Must Use
- `new mongoose.Schema()`
- `mongoose.model()`
- clear field names

## Stretch Goal
Add one required field or default value to the schema.
