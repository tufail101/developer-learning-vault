# Mini Project — Small Typed API Design

## What To Build
Create a small TypeScript API design for one resource such as tasks, books, or notes.

## What It Should Do
- define the main resource type clearly
- use typed request or helper functions for at least two actions
- return typed success or error shapes

## What You Must Use
- resource types
- typed response shapes

## Stretch Goal
Add one shared generic type like `ApiResponse<T>` and reuse it in more than one place.
