# Chapter 3: Git Installation And Setup

## Why Setup Matters

Before using Git smoothly, your machine needs Git installed and your identity configured.

## Basic Setup Commands

Check Git:

```bash
git --version
```

Set your name:

```bash
git config --global user.name "Your Name"
```

Set your email:

```bash
git config --global user.email "your@email.com"
```

## Real-World Analogy

This is like writing your name on your notebook before you start using it.

## Why This Matters

Git uses your name and email in commits so the history shows who made the change.

## Best Practices

- verify Git installation first
- use the same email you want tied to your GitHub account
- check your config when something seems wrong

## Common Mistakes

- skipping the setup step
- using the wrong email
- assuming Git is already installed without checking
