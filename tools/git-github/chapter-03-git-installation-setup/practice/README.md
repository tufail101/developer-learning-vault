# Practice — Git Installation And Setup

## Exercise 1
Run `git --version` and confirm Git works on your machine.

## Exercise 2
Set your username and email with `git config --global` and then run `git config --global --list` to verify both values.

## Exercise 3
Create one practice repository, make one commit, and check that the author name on the commit matches the Git config you just set.
