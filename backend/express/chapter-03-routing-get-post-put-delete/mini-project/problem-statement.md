# Mini Project — Four-Route Notes API

## What To Build
Create a tiny REST-style API for notes using the four main HTTP methods.

## What It Should Do
- Return a list message from `GET /notes`
- Accept a create request at `POST /notes`
- Accept an update request at `PUT /notes/:id`
- Accept a delete request at `DELETE /notes/:id`

## What You Must Use
- `app.get()`
- `app.post()`
- `app.put()`
- `app.delete()`

## Stretch Goal
Return JSON instead of plain text for all four routes.
