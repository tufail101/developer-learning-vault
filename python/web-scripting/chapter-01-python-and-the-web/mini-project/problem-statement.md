# Mini Project — Backend Or Browser?

## What To Build
Build a small Python script that explains which jobs belong to the browser and which jobs belong to the server.

## What It Should Do
- print a heading for browser-side work
- print a heading for server-side work
- list at least three jobs under each heading
- end with one short summary line explaining where Python fits in web development

## What You Must Use
- at least two Python lists
- a loop to print the jobs
- clear printed output, not one giant block of text

## Stretch Goal
Ask the user to type a job like `store data` or `handle button clicks`, then print whether that job belongs on the browser side or the server side.
