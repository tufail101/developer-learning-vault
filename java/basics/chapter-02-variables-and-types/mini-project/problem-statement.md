# Mini Project — Simple Profile Program

## What To Build
Build a small Java program that stores and prints profile information.

## What It Should Do
- store a name, age, and city
- include at least one number and one boolean
- print each value with a clear label

## What You Must Use
- typed variables
- `System.out.println()`

## Stretch Goal
Add a variable for study hours per week and print a complete summary line.
