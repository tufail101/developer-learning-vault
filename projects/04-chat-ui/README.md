# Chat UI

This project is a browser-based chat interface with a contacts sidebar, message bubbles, and simple send/reply behavior.

## Skills It Demonstrates

- full-height app layout
- conversation bubble alignment
- Enter key and submit handling
- auto-scroll to latest message
- generated timestamps
- fake asynchronous reply behavior

## How To Open It

Open [src/index.html](/home/tufail/Dev/Study/web-dev-learning-system/projects/04-chat-ui/src/index.html) in a browser.

## Stretch Goals

- support multiple conversations
- store messages in localStorage
- show “typing…” before the fake reply
- add emoji reactions or attachments
