# Practice — What Is Express

## Exercise 1
Create a new folder, run `npm init -y`, install Express, create `app.js`, and build the smallest server that responds with `Hello from Express` at `/`.

## Exercise 2
Add `"start": "node app.js"` to `package.json`.
Then change the response text so it includes your name or project name.
Run `npm start` and confirm the browser shows the updated message.

## Exercise 3
Add a second route such as `/about` that returns a different message.
Visit both routes in the browser and make sure each one responds correctly.

## Done Checklist
- [ ] I ran `npm install express` before starting the server
- [ ] I added a real `start` script in `package.json`
- [ ] `http://localhost:3000` returns the home response
- [ ] my second route returns different text from the home route
- [ ] I can explain the difference between Node and Express in one or two lines
