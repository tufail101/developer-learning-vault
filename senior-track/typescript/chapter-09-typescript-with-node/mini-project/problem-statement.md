# Mini Project — Typed Config And Data Helpers

## What To Build
Create a small Node-style TypeScript module with typed config values and typed data helper functions.

## What It Should Do
- define a few backend-style types
- use them in functions that transform or read data
- show how typed inputs make the helper functions safer to call

## What You Must Use
- at least two custom types
- typed function parameters and return values

## Stretch Goal
Add one union type for a result like success or failure and handle it safely.
