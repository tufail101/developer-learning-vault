# Mini Project — React Task List From API

## What To Build
Create a React page that fetches a task list from an Express API and renders it on screen.

## What It Should Do
- Fetch data after the component mounts
- Store the result in state
- Render loading, success, and error states clearly

## What You Must Use
- fetching from React
- state updates after data arrives
- loading or error handling

## Stretch Goal
Add a reload button that fetches the list again.
