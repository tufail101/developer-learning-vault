# Chapter 1: Introduction To JavaScript

## What JavaScript Does

JavaScript gives webpages behavior.

HTML creates structure.
CSS creates appearance.
JavaScript handles logic, interaction, and updates.

That means JavaScript can:

- react to clicks
- change page content
- validate forms
- fetch data from APIs
- store data in the browser

## Real-World Analogy

If HTML is the building and CSS is the interior design, JavaScript is the staff inside the building making things happen.

## Why This Matters

Without JavaScript, many modern apps would feel static.
With JavaScript, pages can respond to the user and update in real time.

## First Example

```js
console.log("Hello, JavaScript");
```

This sends a message to the browser console.

## Another Example

```js
alert("Welcome to the page");
```

This shows a popup in the browser.

## Best Practices

- start by understanding what each line does
- use the browser console often
- type code by hand while learning

## Common Mistakes

- trying to memorize everything too early
- copying code without testing it
- not reading console errors carefully

## How JavaScript Works In The Browser

The browser loads your HTML, applies CSS, and then runs JavaScript.
That is why JavaScript can read the page and change it later.
