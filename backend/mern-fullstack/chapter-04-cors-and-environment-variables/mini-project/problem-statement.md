# Mini Project — Configurable API Starter

## What To Build
Set up a small frontend and backend pair that uses CORS and environment variables correctly.

## What It Should Do
- Allow cross-origin requests during local development
- Read at least one value from an environment file
- Avoid hardcoding the most important config values

## What You Must Use
- basic CORS setup
- one environment variable
- keeping config out of hardcoded strings

## Stretch Goal
Use separate frontend and backend environment variables for local API URLs.
