# Mini Project — Typed Admin Views

## What To Build
Create a small TypeScript file that models one full data type and then derives smaller versions for different screens or actions.

## What It Should Do
- start with one base type
- derive at least two new types with utility helpers
- show how each derived type would be used in a realistic UI or API case

## What You Must Use
- at least two utility types
- clear examples of why each derived type exists

## Stretch Goal
Add one type that combines `Pick` with `Readonly` or `Partial` and explain why that version is useful.
