# Chapter 2: Basic Tags

## What A Tag Does

HTML tags tell the browser what kind of content it is reading.

For example:

- a heading is not the same as a paragraph
- important text is not the same as normal text
- a line break is not the same as a new section

That may sound small, but it matters a lot. Browsers, search engines, and screen readers all rely on those signals.

## Real-World Analogy

Think of tags like labels on boxes while moving house.

- a box labeled "Books" tells you what is inside
- a box labeled "Kitchen" tells you how it should be handled

HTML tags do the same for webpage content.

## Common Basic Tags

### Headings

Use headings to create a clear content structure.

```html
<h1>Main Page Title</h1>
<h2>Section Title</h2>
<h3>Smaller Subsection</h3>
```

### Paragraphs

Use paragraphs for blocks of normal text.

```html
<p>This is a paragraph of content.</p>
```

### Emphasis And Importance

```html
<p>I <em>really</em> enjoy building websites.</p>
<p><strong>Important:</strong> Save your work often.</p>
```

## A Small Example

```html
<h1>My Learning Notes</h1>
<h2>Today I Learned</h2>
<p>HTML gives structure to a webpage.</p>
<p><strong>Key idea:</strong> tags describe meaning.</p>
```

## Best Practices

- use only one main `h1` for the main page title
- keep heading order logical
- use paragraphs for content, not headings
- use `strong` and `em` for meaning, not just looks

## Beginner Tip

If you are ever unsure what tag to use, ask yourself:

"What is this content doing?"

That question usually points you to the correct HTML tag.
