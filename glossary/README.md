# Glossary

- Browser: software that reads and shows websites
- DOM: a JavaScript-friendly representation of the page
- Element: an HTML item like a paragraph or button
- Attribute: extra information inside an HTML tag
- Selector: a way to target elements in CSS or JavaScript
- API: a system that lets programs talk to each other
- Responsive: layout that adapts to different screen sizes
- Semantic HTML: HTML that clearly describes meaning
- Accessibility: making websites usable for more people
