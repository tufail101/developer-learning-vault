# Mini Project — Tip Calculator Functions

## What To Build
Build a Python script that uses functions to calculate a restaurant tip and the final bill total.

## What It Should Do
- take a bill amount and tip percentage
- use one function to calculate the tip amount
- use another function to calculate the final total
- print the results in a readable format

## What You Must Use
- at least two functions
- parameters
- return values

## Stretch Goal
Add a third function that splits the final bill between multiple people.
