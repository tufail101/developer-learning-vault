# Mini Project — Public API Data Fetcher

## What To Build
Build a command-line tool that fetches data from a public API and prints a readable summary.

## What It Should Do
- make a request to an API
- parse JSON data
- show a clean result or error message

## What You Must Use
- `requests`
- JSON parsing

## Stretch Goal
Add a command-line menu that lets the user choose between two different endpoints.
