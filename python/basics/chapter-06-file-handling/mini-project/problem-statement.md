# Mini Project — Daily Notes Saver

## What To Build
Build a Python script that saves short daily notes into a text file and then shows all saved notes.

## What It Should Do
- ask for a note with `input()`
- append that note to a file
- read the file back and print all saved notes
- keep older notes instead of overwriting them

## What You Must Use
- file writing
- file reading
- append mode
- a `with open(...)` block

## Stretch Goal
Add the current date before each saved note.
