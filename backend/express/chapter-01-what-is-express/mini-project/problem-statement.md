# Mini Project — First Express Server

## What To Build
Create a tiny Express app that acts like the first draft of a real website backend.

## What It Should Do
- Start an Express server on a local port
- Return one message at `/` and a different message at `/about`
- Log a clear startup message in the terminal

## What You Must Use
- installing Express with npm
- creating an app with `express()`
- at least two basic routes with `app.get()`

## Stretch Goal
Add a `/contact` route with a third response.
