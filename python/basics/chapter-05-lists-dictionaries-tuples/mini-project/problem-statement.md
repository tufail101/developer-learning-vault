# Mini Project — Study Planner Data

## What To Build
Build a small script that stores study data using a list, a dictionary, and a tuple.

## What It Should Do
- store a list of study topics
- store learner details in a dictionary
- store one fixed pair of values in a tuple

## What You Must Use
- one list
- one dictionary

## Stretch Goal
Loop through the list of topics and print each one with its position.
