# Practice — Node Modules

## Exercise 1
Create a file named `greetings.js` that exports one function returning `Hello, learner!`.

## Exercise 2
Create an `app.js` file that imports the function from `greetings.js` and logs the returned message.

## Exercise 3
Add a second exported function that takes a name as an argument. Import and run both functions from `app.js` so you can see the difference between a fixed message and a dynamic one.
