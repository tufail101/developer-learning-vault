# Practice — What Is Express

## Exercise 1
Install Express in a fresh practice folder and create the smallest server that responds with `Hello from Express` at `/`.

## Exercise 2
Change the response text so it includes your name or project name, then restart the server and test it in the browser.

## Exercise 3
Add one second route such as `/about` with a different message so you can prove Express is routing requests correctly.
