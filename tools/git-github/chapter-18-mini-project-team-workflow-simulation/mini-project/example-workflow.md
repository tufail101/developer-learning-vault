# Example Workflow

1. Initialize a new repository and make the first commit on `main`.
2. Push the repository to GitHub.
3. Create a feature branch like `feature/add-readme-section`.
4. Make one small change and commit it with a focused message.
5. Push the feature branch and open a pull request.
6. Leave one review note or checklist item.
7. Update the branch if needed.
8. Merge the pull request and confirm the branch is complete.
