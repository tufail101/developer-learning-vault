# Mini Project — First Repo Automation

## What To Build
Create a small GitHub Actions workflow that runs one simple check when code is pushed.

## What It Should Do
- live inside `.github/workflows/`
- trigger on a clear event like `push` or `pull_request`
- run one simple automated step

## What You Must Use
- a workflow YAML file
- at least one job and one step

## Stretch Goal
Add a second step that checks formatting, runs tests, or prints a clearer project status message.
