# Dashboard UI

This project is a responsive analytics dashboard with a sidebar, stat cards, activity table, and section switching.

## Skills It Demonstrates

- app-style page structure
- CSS Grid layout for dashboard sections
- responsive sidebar behavior
- tab switching with JavaScript
- animated count-up stats
- card, chart placeholder, and table styling

## How To Open It

Open [src/index.html](/home/tufail/Dev/Study/web-dev-learning-system/projects/03-dashboard-ui/src/index.html) in a browser.
