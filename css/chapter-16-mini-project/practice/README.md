# Practice — CSS Mini Project Prep

## Exercise 1
Choose a page section, such as a hero or pricing area, and sketch its layout using simple boxes before coding.

## Exercise 2

## Exercise 3
Add one responsive adjustment so the section still reads clearly on a smaller screen.
