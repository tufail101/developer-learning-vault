# Mini Project — Django Starter Map

## What To Build
Build a small starter guide file for yourself called `django-starter-map.md` or `django-starter-map.txt`.
This is not a web app yet.
It is a personal map that helps you stop feeling lost when you create your first Django project.

## What It Should Do
- explain in plain words what Django is
- list the commands to create and run a new Django project
- name the difference between a Django project and a Django app
- list at least three built-in tools Django gives you

## What You Must Use
- your own words, not copied chapter text
- one short Django vs Flask comparison
- one section that lists the setup commands in order

## Stretch Goal
Add a short section called `When I would pick Django` with two real project examples.
