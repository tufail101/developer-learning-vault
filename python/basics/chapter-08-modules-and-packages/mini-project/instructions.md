# Instructions — Reusable Helpers Folder

1. Create a main file such as `app.py`.
2. Create a helper file such as `helpers.py`.
3. Add one or two reusable functions inside `helpers.py`.
4. Import those functions into `app.py`.
5. Call them from `app.py` and print the results.
6. Run `python3 app.py` to confirm the import works.

## Suggested Build Order

1. Write the helper function first.
2. Test its name and output.
3. Add the import line in `app.py`.
4. Call the function from the main file last.

## What To Check Before You Stop

1. Does `app.py` run without copying the helper code into it?
2. Is the helper file holding logic that would otherwise clutter the main file?
3. Are the file names clear and not clashing with built-in module names?
