# Mini Project — Course Creator Script

## What To Build
Write a small script that creates course documents and then reads them back from the database.

## What It Should Do
- Create at least two course documents
- Read all courses back from the collection
- Log the results so you can confirm the save worked

## What You Must Use
- `Model.create()`
- `Model.find()`
- async database operations

## Stretch Goal
Sort the courses alphabetically before logging them.
