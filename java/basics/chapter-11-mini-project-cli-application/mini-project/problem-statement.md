# Mini Project — Command Line Task Tracker

## What To Build
Build a small Java command-line app that helps a user track simple tasks.

## What It Should Do
- show a menu in the terminal
- let the user add or view tasks
- keep the code split into readable methods

## What You Must Use
- methods
- basic collections or arrays

## Stretch Goal
Save the tasks to a file so they still exist on the next run.
