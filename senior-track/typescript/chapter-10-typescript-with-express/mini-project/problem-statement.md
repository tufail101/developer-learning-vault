# Mini Project — Typed Route Helpers

## What To Build
Create a small set of Express-style TypeScript helpers for listing, creating, or updating one resource.

## What It Should Do
- type at least one params object and one request body
- return a predictable typed response shape
- show how the typed shapes guide each helper function

## What You Must Use
- params or body types
- typed helper functions

## Stretch Goal
Add a result type for validation errors so your route helper can describe failure clearly too.
