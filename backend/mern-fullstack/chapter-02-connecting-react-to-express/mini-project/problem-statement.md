# Mini Project — Frontend To Backend Greeting

## What To Build
Connect a small frontend view to one Express route and render the returned message.

## What It Should Do
- Request data from the backend
- Receive JSON successfully
- Show the result on screen

## What You Must Use
- one Express JSON route
- one frontend request to that route
- matching local URLs

## Stretch Goal
Add a loading message while the request is in progress.
