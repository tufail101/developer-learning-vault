# Mini Project — Study Planner Data

## What To Build
Build a small script that stores study planner data using a list, a dictionary, and a tuple.

## What It Should Do
- store a list of study topics
- store learner details in a dictionary
- store one fixed pair of values in a tuple, such as start time and end time or screen width and height
- print all of that data in a readable way

## What You Must Use
- one list
- one dictionary
- one tuple

## Stretch Goal
Loop through the list of topics and print each one with its position number.
