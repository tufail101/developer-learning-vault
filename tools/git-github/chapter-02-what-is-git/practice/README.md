# Practice — What Git Is

## Exercise 1
Create a small file, commit it once, then change one line and run `git status` to see that Git notices the change.

## Exercise 2
Make a second commit after the change and run `git log --oneline` to confirm there are now two saved points in history.

## Exercise 3
Open both commit messages and describe what changed between them in one sentence each.
