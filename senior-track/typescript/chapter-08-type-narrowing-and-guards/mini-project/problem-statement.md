# Mini Project — Safe Response Handler

## What To Build
Create a TypeScript file that handles multiple possible API response shapes without unsafe assumptions.

## What It Should Do
- define at least two response shapes
- narrow the type before reading shape-specific fields
- print different output for success and error cases

## What You Must Use
- a union type
- one built-in or custom type guard

## Stretch Goal
Add a third response state such as loading or empty and handle it safely too.
