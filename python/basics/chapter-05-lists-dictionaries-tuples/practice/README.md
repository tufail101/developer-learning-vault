# Practice — Lists Dictionaries Tuples

## Exercise 1
Create a list of five foods.
Print the second item.
Then add one more food with `append()` and print the full list.

## Exercise 2
Create a dictionary for a user with `name`, `email`, and `city`.
Print each value using its key.

## Exercise 3
Create a tuple for screen width and height.
Print a sentence that uses both values, such as `"The window size is 1280 by 720."`
