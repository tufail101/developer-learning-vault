# Practice — Node.js Introduction

## Exercise 1
Create a file named `hello.js` that logs `Hello from Node.js` to the terminal. Run it with `node hello.js` and make sure the message prints once.

## Exercise 2
Update the file so it logs your name and one reason you want to learn backend development. Run it again and check that both lines appear.

## Exercise 3
Create a second file named `math.js` that logs the result of `5 + 3`, `10 - 4`, and `6 * 7`. Run the file and verify all three results in the terminal.
