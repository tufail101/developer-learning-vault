# Mini Project — MERN Auth Starter App

## What To Build
Create a complete small MERN app with registration, login, and one protected feature.

## What It Should Do
- Register a new user
- Log in an existing user and receive a token
- Show one protected feature only after successful login
- Store user records in MongoDB through Mongoose

## What You Must Use
- frontend to backend API calls
- MongoDB or Mongoose for stored data
- registration and login flow
- one protected feature

## Stretch Goal
Add a logout button that clears the saved token and hides the protected content again.
