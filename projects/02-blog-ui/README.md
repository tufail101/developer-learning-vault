# Blog UI

This project is an editorial-style blog interface with a featured article, post cards, categories, tags, and filtering tools.

## Skills It Demonstrates

- semantic content layout
- CSS Grid for multi-column structure
- card hover effects
- typography scale and hierarchy
- category filtering with JavaScript
- live search by post title

## How To Open It

Open [src/index.html](/home/tufail/Dev/Study/web-dev-learning-system/projects/02-blog-ui/src/index.html) in a browser.
