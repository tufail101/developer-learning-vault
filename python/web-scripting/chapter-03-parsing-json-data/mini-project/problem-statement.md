# Mini Project — Todo JSON Reader

## What To Build
Build a script that fetches a todo item from an API and prints a clean summary from the JSON response.

## What It Should Do
- read JSON from a response
- extract at least two fields
- print a readable sentence

## What You Must Use
- `response.json()` or `json.loads()`
- dictionary key access

## Stretch Goal
Loop through a list endpoint and print the first five titles instead of one item.
