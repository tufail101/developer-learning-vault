# Mini Project — Undo Command Lab

## What To Build
Create a safe throwaway repository where you practice the main Git undo commands and compare their results.

## What It Should Do
- use `git restore` on local file changes
- use `git revert` to undo a commit without rewriting shared history
- use one form of `git reset` locally and describe what changed

## What You Must Use
- `git restore`
- `git revert` or `git reset` used with clear intent

## Stretch Goal
Write a short note explaining when you would choose `revert` instead of `reset` on a team project.
