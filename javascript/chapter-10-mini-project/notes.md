# Chapter 10: JavaScript Mini Project

## Goal

Bring together what you learned so far:

- variables
- conditions
- functions
- arrays
- DOM updates
- events

## Suggested Ideas

- to-do app
- quiz app
- note app
- product filter
- counter app

## Why This Matters

This is where JavaScript stops feeling like separate small lessons and starts feeling like real frontend development.

## Best Practices

- break the project into small steps
- get one feature working at a time
- test after every change
- keep functions focused

## Common Mistakes

- trying to build the whole app at once
- mixing UI code and data logic too much
- skipping small tests during development
