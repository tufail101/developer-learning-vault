# Practice — Undoing Things Reset Revert Restore

## Exercise 1
Make a small commit in a test repo, then use `git revert` to undo it and inspect the new commit history.

## Exercise 2
Edit a tracked file without committing, then use `git restore` to bring it back to the last committed version.

## Exercise 3
Create two local commits in a test repo and practice `git reset --soft HEAD~1` so you can see how the commit history changes while your file changes stay staged.
