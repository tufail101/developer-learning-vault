# Roadmap

## Step 1 HTML

Learn how web pages are structured.

- tags
- nesting
- forms
- tables
- semantic sections
- accessibility basics

## Step 2 CSS

Learn how to style and layout the page.

- selectors
- box model
- spacing
- colors
- typography
- Flexbox
- Grid
- responsive design

## Step 3 JavaScript

Learn how to add logic.

- variables
- data types
- conditions
- loops
- functions
- arrays
- objects

## Step 4 DOM And APIs

Learn how JavaScript interacts with the page and external data.

- DOM selection
- events
- forms
- fetch API
- async and await
- error handling

## Step 5 Projects

Build polished UI work that shows your skills.

- portfolio
- blog
- dashboard
- chat UI
- ecommerce UI

## 3-Year Journey

### Year 1

- Build fundamentals
- Repeat basics often
- Learn to debug calmly

### Year 2

- Build medium projects
- Learn APIs and data flow
- Improve code organization

### Year 3

- Polish portfolio
- Learn collaboration habits
- Prepare for interviews and freelance/client work
