# Mini Project — Simple Dashboard Layout

## What To Build
Create a page layout with a header and a section of cards using modern layout tools.

## What It Should Do
- Use Flexbox for one part of the page such as navigation
- Use Grid for the main card area
- Keep the spacing even and easy to scan

## What You Must Use
- Flexbox
- Grid
- consistent spacing

## Stretch Goal
Add a media query so the cards stack on smaller screens.
