# Mini Project — Split Route API

## What To Build
Refactor a small Express app so related routes live in their own router file.

## What It Should Do
- Keep the main server file short and readable
- Move one route group into a router module
- Return JSON from the router routes

## What You Must Use
- `express.Router()`
- separate route file
- `app.use()` to mount the router

## Stretch Goal
Create two routers, such as `users` and `products`, and mount both in the main app.
