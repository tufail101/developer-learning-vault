# Chapter 6: GitHub Introduction

## What GitHub Is

GitHub is a platform for storing Git repositories online.

It helps developers:

- back up code online
- share projects
- collaborate with others
- review changes

## Real-World Analogy

If Git is your local notebook, GitHub is the shared cloud shelf where that notebook can be stored and shared.

## Why This Matters

GitHub is a major part of modern developer workflow.

Recruiters, teams, open-source communities, and collaborators often use GitHub every day.

## Common GitHub Actions

- create repository
- push local code
- open pull request
- review code

## Best Practices

- give repositories clear names
- write helpful repository descriptions
- keep your README useful

## Common Mistakes

- confusing local Git history with remote GitHub state
- pushing the wrong project
- forgetting to connect the local repo to GitHub
