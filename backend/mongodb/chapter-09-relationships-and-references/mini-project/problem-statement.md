# Mini Project — Course And Lesson Relationship Plan

## What To Build
Model a simple relationship between courses and lessons using Mongoose references.

## What It Should Do
- Create a course schema
- Create a lesson schema that references the course
- Keep the relationship easy to understand from the code

## What You Must Use
- ObjectId references
- `ref` in a schema
- the idea of linking documents

Add one note about when embedding might be simpler than referencing.
