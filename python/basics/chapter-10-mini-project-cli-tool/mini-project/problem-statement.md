# Mini Project — Command Line Study Tracker

## What To Build
Build a small command-line study tracker that lets the user add and view study tasks.

## What It Should Do
- show a menu in the terminal
- let the user add tasks
- let the user view saved tasks during the same run

## What You Must Use
- functions
- lists or dictionaries

## Stretch Goal
Save tasks to a file so they are still there the next time the script runs.
