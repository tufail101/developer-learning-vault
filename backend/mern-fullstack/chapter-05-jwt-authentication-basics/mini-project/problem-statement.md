# Mini Project — Token Flow Plan

## What To Build
Plan the first version of a login flow that uses JWTs between a React frontend and Express backend.

## What It Should Do
- Include a login request
- Include a token response
- Include one later protected request that sends the token back

## What You Must Use
- login flow idea
- token returned by the backend
- Authorization header on later requests

## Stretch Goal
Add a logout note describing what the frontend should clear when the user signs out.
