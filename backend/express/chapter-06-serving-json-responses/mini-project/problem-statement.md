# Mini Project — Course JSON API

## What To Build
Create a small JSON API that returns course data a frontend could use later.

## What It Should Do
- Return one course list route
- Return one single-course route
- Use JSON objects with clear field names

## What You Must Use
- `res.json()`
- objects and arrays in responses
- clear property names

## Stretch Goal
Add a `published` boolean to each course and filter with a query string.
