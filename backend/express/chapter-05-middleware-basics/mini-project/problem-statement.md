# Mini Project — Request Logger App

## What To Build
Create a small Express app that uses middleware to log and enrich incoming requests.

## What It Should Do
- Log every request method and path
- Add one custom value to the request object
- Return that custom value from at least one route

## What You Must Use
- custom middleware function
- `next()`
- a route that runs after middleware

## Stretch Goal
Add one middleware that blocks a request when a query value is missing.
