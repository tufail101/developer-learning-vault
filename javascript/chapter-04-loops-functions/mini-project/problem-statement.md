# Mini Project — Repeating Greeting Generator

## What To Build
Create a small page or script that generates multiple greeting messages from a list of names.

## What It Should Do
- Store several names in an array
- Use a function to format each greeting
- Use a loop to show every greeting

## What You Must Use
- one reusable function
- one loop
- visible or console output for each name

## Stretch Goal
Add a second function that counts how many greetings were created.
