# Mini Project — Dynamic Task Board

## What To Build
Create a small UI where the user can add tasks to the page dynamically.

## What It Should Do
- Read text from an input
- Create a new task element with JavaScript
- Append the new task to a visible list and style it with a class

## What You Must Use
- `createElement()`
- `appendChild()` or `append()`
- `classList` updates

## Stretch Goal
Add a remove or complete button for each task.
