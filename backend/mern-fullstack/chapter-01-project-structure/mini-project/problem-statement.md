# Mini Project — MERN Project Skeleton

## What To Build
Plan the first folder structure for a small MERN app before adding features.

## What It Should Do
- Separate the frontend and backend clearly
- Include room for routes, models, and components
- Keep the layout easy to understand at a glance

## What You Must Use
- separate frontend and backend folders
- clear naming
- a simple mental map of the app

## Stretch Goal
Add a short README that explains the purpose of each top-level folder.
