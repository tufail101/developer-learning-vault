# Mini Project — Shell Command Cheat Sheet

## What To Build
Create a short practice sheet of MongoDB shell commands for one pretend app database.

## What It Should Do
- Switch to a database
- Insert at least two documents
- Read the documents back with a find command

## What You Must Use
- `use`
- `insertOne()`
- `find()`

## Stretch Goal
Add one `findOne()` command for a specific document.
