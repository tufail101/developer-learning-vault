# Python Track

This track adds Python as another path in the learning system.
It starts with basics, then moves into web scripting, Flask, and a first look at Django.

## Modules

- `basics/`: core Python syntax, files, errors, and a small command-line project
- `web-scripting/`: requests, JSON, and HTML parsing basics
- `flask/`: routes, templates, forms, databases, APIs, and simple auth ideas
- `django/`: project structure, models, views, templates, and API basics
