# Chapter 11: SEO Basics And Meta Tags

## What SEO Means

SEO stands for Search Engine Optimization.

At a beginner level, this mostly means helping search engines understand what your page is about.

HTML helps with this more than many beginners realize.

## What Meta Tags Do

Meta tags live inside the `head` section and provide information about the page.

Common examples:

- page title
- description
- viewport settings
- character encoding

## Real-World Analogy

Imagine giving a book a proper title, summary, and category before placing it in a library.
Meta tags do something similar for a webpage.

## Example

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta
    name="description"
    content="A beginner-friendly HTML learning page with examples and exercises."
  />
  <title>Learn HTML Basics</title>
</head>
```

## Why Titles Matter

The page title:

- appears in the browser tab
- helps users understand the page
- helps search engines understand the topic

Bad title:

```html
<title>Home</title>
```

Better title:

```html
<title>HTML Basics Guide For Beginners</title>
```

## Best Practices

- write a clear and specific title
- add a useful meta description
- keep the title related to the page content
- do not stuff keywords unnaturally

## Mentor Tip

SEO is not magic. Start with clarity. Clear titles, clear structure, and clear content are already strong beginner habits.
