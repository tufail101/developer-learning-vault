# Practice — Gitignore And Git Clean

## Exercise 1
Create a `.gitignore` file that ignores `node_modules/`, `.env`, and a `dist/` folder.

## Exercise 2
Create a fake untracked file like `debug.log`, run `git clean -n`, and confirm Git previews it without deleting it yet.

## Exercise 3
Add one tracked file and one ignored file to a test repo, then compare what `git status` shows for each one.
