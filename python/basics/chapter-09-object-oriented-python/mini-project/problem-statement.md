# Mini Project — Library Book Class

## What To Build
Build a small class-based Python script for tracking library books.

## What It Should Do
- create a `Book` class
- store title and author on each object
- add a method that returns a readable summary
- create at least two different book objects

## What You Must Use
- a class
- `__init__`
- at least one method

## Stretch Goal
Add a method that marks a book as borrowed or available.
