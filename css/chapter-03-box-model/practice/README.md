# Practice — Box Model

## Exercise 1
Create one box with a fixed width, padding, border, and margin.

## Exercise 2
Build a second box beside it and change the margin so you can see how spacing affects layout.

## Exercise 3
Turn on `box-sizing: border-box` and compare how the final width behaves before and after the change.
