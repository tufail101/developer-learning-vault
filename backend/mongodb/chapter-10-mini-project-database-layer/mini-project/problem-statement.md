# Mini Project — Learning App Database Layer

## What To Build
Create a small MongoDB and Mongoose database layer for a learning app resource.

## What It Should Do
- Connect to the database
- Define at least one model
- Create, read, update, and delete documents
- Keep the code organized enough that an Express API could use it next

## What You Must Use
- Mongoose connection
- at least one model
- create/read/update/delete flow

## Stretch Goal
Add a second model with a relationship or reference to the first one.
