# HTML Cheatsheet

- `<!DOCTYPE html>` starts a modern HTML document
- `<head>` stores metadata
- `<body>` stores visible page content
- `<h1>` to `<h6>` are headings
- `<p>` is paragraph text
- `<a>` creates a link
- `<img>` displays an image
- `<ul>` and `<ol>` create lists
- `<table>` displays tabular data
- `<form>` collects user input
- `<header>`, `<main>`, `<section>`, `<article>`, `<footer>` give semantic meaning
