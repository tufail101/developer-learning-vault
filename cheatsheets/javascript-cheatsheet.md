# JavaScript Cheatsheet

- `let` creates a block-scoped variable
- `const` creates a value that should not be reassigned
- `if` runs code conditionally
- `for` repeats logic
- `function` defines reusable behavior
- arrays store ordered values
- objects store key-value pairs
- `document.querySelector()` selects a DOM element
- `addEventListener()` listens for user actions
- `fetch()` requests data from an API
