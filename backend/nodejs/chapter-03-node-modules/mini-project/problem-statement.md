# Mini Project — Reusable Utility Module

## What To Build
Create a small Node.js app that imports functions from another file instead of writing everything in one place.

## What It Should Do
- Export at least two utility functions from one module file
- Import those functions into an app file
- Run the app and print both results in the terminal

## What You Must Use
- module exports
- imports with `require()` or the chapter approach
- clear separation between the utility file and the main file

## Stretch Goal
Add a third helper that accepts user input from the command line.
