# Mini Project — First Java Welcome Program

## What To Build
Build a tiny Java program that greets the user and prints a short learning message.

## What It Should Do
- print a welcome line
- print your name or learner name
- print one simple study reminder

## What You Must Use
- a `main` method
- `System.out.println()`

## Stretch Goal
Add a second class with a helper method that prints the reminder.
