# Mini Project — Feedback Form Handler

## What To Build
Build a small Flask form flow that collects a name and a short message.

## What It Should Do
- show a form
- accept a POST request
- read `name` and `message` from the submitted form
- display a thank-you message after submission
- show an error message if either field is empty

## What You Must Use
- form handling
- `request.form`
- `methods=["GET", "POST"]`

## Stretch Goal
Show an error message when the name or message is empty.
