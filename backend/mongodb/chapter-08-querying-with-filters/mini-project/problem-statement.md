# Mini Project — Filtered Course Search

## What To Build
Create a script that returns only the documents matching a small set of search rules.

## What It Should Do
- Filter by at least one field
- Filter by at least two fields together
- Log the final filtered result clearly

## What You Must Use
- filter objects in `find()`
- matching on field values
- reading only useful documents

## Stretch Goal
Add a sort step after the filter so the results come back in a predictable order.
