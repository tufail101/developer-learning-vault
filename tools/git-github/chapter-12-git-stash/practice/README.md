# Practice — Git Stash

## Exercise 1
Make an uncommitted change in a test repo, stash it with a message, and confirm it disappears from `git status`.

## Exercise 2
Run `git stash list` and identify the stash entry you just created.

## Exercise 3
Use `git stash pop` to restore the work and confirm the file changes return.
