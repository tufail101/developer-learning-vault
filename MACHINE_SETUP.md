# Machine Setup

## 1. Install VS Code

- Download Visual Studio Code from the official website
- Install it with default settings
- Recommended extensions:
  - HTML CSS Support
  - Live Server
  - Prettier
  - ESLint
  - GitLens

## 2. Install Node.js

- Download the LTS version of Node.js
- Verify installation:

```bash
node -v
npm -v
```

Why install Node.js if we are learning frontend first?

Because modern web development often uses package tools, local servers, formatters, and build scripts later in the journey.

## 3. Learn Browser Dev Tools

Open Chrome or Edge and practice:

- Inspect element
- View console
- Check network requests
- Edit styles live
- Test responsive layouts

## 4. Install Git

Verify installation:

```bash
git --version
```

Set your name and email:

```bash
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
```

## 5. Create GitHub Account

- Make a GitHub account
- Create a practice repository
- Push your first HTML file
- Learn `git clone`, `git add`, `git commit`, and `git push`

## 6. Folder Habit

Keep a clean workspace:

- `learning/` for chapter practice
- `projects/` for finished builds
- `notes/` for personal summaries
