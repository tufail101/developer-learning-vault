# Mini Project — Reusable Helpers Folder

## What To Build
Build a tiny multi-file Python project with one main file and one helper module.

## What It Should Do
- store reusable functions in a helper file
- import those functions into the main file
- run the main file successfully

## What You Must Use
- imports
- at least two Python files

## Stretch Goal
Add a second helper module for formatting text or numbers.
