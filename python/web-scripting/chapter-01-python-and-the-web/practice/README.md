# Practice — Python And The Web

## Exercise 1
Write a script called `web_roles.py`.
Make it print three jobs Python can do on the server side of a web project.

## Exercise 2
Create two lists:
`browser_jobs` and `server_jobs`.
Put at least three items in each list.
Print both lists with clear labels so you can see the difference in the terminal.

## Exercise 3
Write a script that prints this idea in your own words:
Python helps build the backend.
JavaScript handles code that runs in the browser.
Do not copy that sentence exactly.
Rewrite it like you are explaining it to a friend.
