# Instructions — Personal Profile Script

1. Create a file such as `profile.py`.
2. Add variables for your name, age, city, and learning goal.
3. Add one boolean variable such as `is_studying_today`.
4. Use `print()` to show each value with a clear label like `"Name:"` or `"City:"`.
5. Run the file with `python3 profile.py`.
6. Change one value and run the file again to confirm the output updates.

## Suggested Build Order

1. Add the text values first.
2. Add the age number second.
3. Add the boolean value third.
4. Write the print lines last.

## What To Check Before You Stop

1. Did you use variables for every piece of profile data?
2. Is at least one value a number and one value a boolean?
3. Does the output read clearly in the terminal?
