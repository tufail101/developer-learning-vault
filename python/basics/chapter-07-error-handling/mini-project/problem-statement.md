# Mini Project — Safe Number Checker

## What To Build
Build a script that asks for two numbers and safely handles invalid input.

## What It Should Do
- ask the user for two values
- convert them to numbers
- print the sum or a clear error message

## What You Must Use
- `try` and `except`
- at least one specific exception type

## Stretch Goal
Let the user try again instead of ending after one bad input.
