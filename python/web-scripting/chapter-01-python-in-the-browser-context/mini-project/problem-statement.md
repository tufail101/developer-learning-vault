# Mini Project — Web Role Explainer

## What To Build
Build a script that prints a simple explanation of how Python and JavaScript work together in web projects.

## What It Should Do
- list at least three Python jobs
- list at least three browser-side jobs
- print a short summary sentence at the end

## What You Must Use
- lists
- a loop

## Stretch Goal
Turn the explanation into a function that takes a project type and prints a custom message.
