# Instructions — Todo JSON Reader

1. Create a file named `todo_json_reader.py`.
2. Import `requests`.
3. Send one request to the JSONPlaceholder todo endpoint.
4. Convert the response to Python data with `response.json()`.
5. Print the `id`, `title`, and `completed` values in one readable sentence.
6. Add simple error handling so the script prints a message if the request fails.

## Suggested Build Order

1. Get the request working first.
2. Print the whole parsed JSON once so you can inspect the shape.
3. Replace that with a cleaner summary sentence.
4. Add error handling.
5. Try the five-todo stretch goal only after the first version is working.
