# Practice — Connecting To A Database Sqlite

## Exercise 1
Create a SQLite database file and one simple table like `notes` with `id` and `title`.

## Exercise 2
Insert one row into the table and then query it back with a `SELECT` statement.
Print the result so you can confirm the row was actually saved.

## Exercise 3
Connect that database setup to a Flask route that reads rows from the table and returns them.
Even plain text output is fine for this exercise.

## Done Checklist
- [ ] My database file is actually created on disk
- [ ] My table exists and I inserted at least one row into it
- [ ] My `SELECT` query returns the row I inserted
- [ ] I used `commit()` after writing data
