# Mini Project — Safe Number Checker

## What To Build
Build a Python script that asks the user for two numbers and handles invalid input without crashing.

## What It Should Do
- ask the user for two values
- try to convert both values into numbers
- print the sum if both values are valid
- print a clear error message if one of them is invalid

## What You Must Use
- `try`
- `except`
- at least one specific exception type such as `ValueError`

## Stretch Goal
Let the user try again instead of ending after one bad input.
