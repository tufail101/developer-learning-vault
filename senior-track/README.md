# Senior Track

Year 3 is where the repo shifts from "I can build things" to "I can build things professionally and explain my decisions clearly."

This track focuses on:

- React fundamentals for component-driven apps
- TypeScript for safer, more maintainable code
- testing for confidence and refactoring safety
- career preparation for interviews, portfolios, and team communication

## Sections

- `react/`: modern component-based frontend thinking
- `typescript/`: type safety and tooling
- `testing/`: unit, integration, and DOM testing
- `career/`: interview prep, system design basics, PR review, portfolio, salary negotiation
