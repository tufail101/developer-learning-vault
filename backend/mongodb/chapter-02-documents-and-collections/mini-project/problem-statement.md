# Mini Project — Collection Map

## What To Build
Plan the main collections for a small app before touching the database itself.

## What It Should Do
- List at least three collections your app would need
- Write one example document for each collection
- Keep the document purpose clear for each collection

## What You Must Use
- the ideas of collections and documents
- consistent document purpose inside one collection

## Stretch Goal
Add one note explaining which collection would grow the fastest over time.
