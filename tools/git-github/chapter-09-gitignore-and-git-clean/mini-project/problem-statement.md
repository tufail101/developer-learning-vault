# Mini Project — Safe Repo Cleanup

## What To Build
Set up a small practice repository where you ignore local clutter and clean up untracked files safely.

## What It Should Do
- use a `.gitignore` file for common junk files
- preview untracked file cleanup before deleting anything
- keep only the files you actually want Git to care about

## What You Must Use
- a `.gitignore` file
- `git clean -n` before cleanup

## Stretch Goal
Add one example of a file that should stay local on your machine and explain why it belongs in `.gitignore`.
