# Variables And Data Types

## What This Chapter Is About

This chapter teaches saving values in variables and working with strings, numbers, and booleans.
The goal is to help you understand the shape of the idea before you worry about bigger projects.

## Real-World Analogy

Variables are labeled storage boxes. The label helps you find the value again later.

## Key Ideas

- Variables store values you want to reuse.
- Common Python data types include `str`, `int`, `float`, and `bool`.
- Good variable names make code easier to read.

## Example

```py
name = "Aisha"
age = 24
is_learning = True

print(name)
print(age)
print(is_learning)
```

## How To Think About It In Practice

When you are building real things, this idea matters because small pieces need to connect clearly.
If the basic step is confusing, later chapters feel much heavier than they need to.
A good habit is to run the example, change one line, and watch what changes.

## Common Mistakes

- using vague names like `data` or `thing`
- mixing strings and numbers without converting them first
- overwriting a useful variable too early
