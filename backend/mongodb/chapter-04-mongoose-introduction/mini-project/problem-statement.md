# Mini Project — First Mongoose Connection

## What To Build
Set up a small Node.js project that connects to MongoDB through Mongoose.

## What It Should Do
- Install and require Mongoose
- Connect to a local database
- Log a clear success or failure message

## What You Must Use
- `mongoose.connect()`
- the idea of schemas and models
- using Mongoose from Node.js

## Stretch Goal
Move the connection string into a variable at the top of the file so it is easier to change later.
