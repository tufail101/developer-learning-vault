# Mini Project — Script-Driven Starter Project

## What To Build
Set up a small Node.js project with a useful `package.json` and runnable npm scripts.

## What It Should Do
- Include a valid `package.json` file
- Add at least a `start` script and one extra custom script
- Run the scripts successfully from the terminal

## What You Must Use
- `npm init` or an equivalent package file
- custom npm scripts
- one installed dependency or dev dependency

## Stretch Goal
Add a README that explains what each npm script does and note that Express is the next backend step.
