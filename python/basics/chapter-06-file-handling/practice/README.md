# Practice — File Handling

## Exercise 1
Create a file named `todo.txt` and write three tasks into it using write mode.

## Exercise 2
Read `todo.txt` back and print its contents in the terminal.

## Exercise 3
Append one more task to `todo.txt` using append mode.
Then read the file again and confirm the new task appears at the end instead of replacing the old lines.

## Done Checklist
- [ ] I used `"w"` when I wanted to create or replace file content
- [ ] I used `"a"` when I wanted to add to the end of the file
- [ ] my file output includes separate lines instead of one squashed block
- [ ] I used a `with open(...)` block instead of leaving the file open manually
